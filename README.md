# Carrot_Game
**javascript를 사용해 미니게임을 만들었습니다.  
제한 시간 안에 당근을 모두 클릭하지 못하거나, 벌레를 클릭할 시 게임이 종료됩니다.  
상단의 정지 버튼을 누르면 게임을 다시 시작할 수 있습니다**
